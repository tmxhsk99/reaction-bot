해석 가이드

  • blank: true + mean=0 stddev=0 → 캡처 자체가 검정. OBS 미연결 / 잘못된 씬 / 모니터 인덱스 문제.
  • blank: true인데 이미지는 보임 → blank-luma-stddev 가 너무 높음. /config 에서 낮추거나 0 으로 비활성.
  • blank: false + 이미지 OK → 캡처 정상. LLM 단계 문제. /api/translate/state 응답 확인.
  • dhashHex 가 매번 동일 → 화면이 안 바뀜(또는 캡처가 정지). 같은 hash 면 LLM 호출 자체 스킵됨.
