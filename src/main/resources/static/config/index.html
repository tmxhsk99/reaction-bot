LLM Provider 필수

실제 응답을 생성할 LLM. provider별로 필요한 키/경로가 달라요 — 아래에서 선택하면 해당 섹션만 펼쳐집니다.
발급: console.anthropic.com/settings/keys · 비워두면 환경변수 ANTHROPIC_API_KEY 사용.
발급: aistudio.google.com/apikey · 비워두면 환경변수 GEMINI_API_KEY 사용.
발급: platform.openai.com/api-keys · 비워두면 환경변수 OPENAI_API_KEY 사용.
자동 탐색 동작: Windows 기본 위치 %APPDATA%\Claude\claude-code\<버전>\claude.exe 에서 가장 높은 버전을 자동 선택. PATH에 claude가 있어도 동작. 자동 탐색이 안 맞으면 아래에 수동 경로 지정.
빈 값 = 기본값 claude(PATH 탐색) 사용. 자동 탐색이 성공하면 이 값은 무시됨.
npm 글로벌 설치 등 비표준 위치면 여기에 디렉토리를, 그리고 위 실행 파일 경로에 절대경로를 직접 적기. 사전 claude 로그인 필수.
자동 탐색 동작: PATH에 등록된 codex 실행파일 사용. PATH에 없으면 아래에 절대 경로 직접 입력. 인증은 codex login(구독, 토큰 0) 또는 아래 API 키(종량) 둘 중 하나.
빈 값 = codex (PATH 탐색). 환경변수 CODEX_CLI_EXEC로도 덮어쓰기 가능.
설정 시 CODEX_API_KEY로 전달 (종량 과금). 비워두면 codex login의 저장된 구독 인증 사용.
로컬 Ollama 서버. ollama pull qwen3-vl:8b 후 사용. 서버는 http://localhost:11434 기본. 상세 옵션은 src/main/resources/ollama.yml 직접 편집.
